Mono uppercase role label above a headline; use it to name the slide's job (not to state a fact).

```jsx
<Kicker>Defensive Move</Kicker>
<Kicker accent="#39ff88" rule={false}>Myth vs Reality</Kicker>
```

Always pairs with a `Headline` below it. Reads the inherited `--accent`, so wrap the slide in a `theme-*` class to recolour. Keep it short — two or three words.
