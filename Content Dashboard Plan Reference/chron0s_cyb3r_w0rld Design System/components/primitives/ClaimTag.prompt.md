The trust-standard credibility badge — labels a claim's altitude so the account stays "no fake panic".

```jsx
<ClaimTag tag="verified" />
<ClaimTag tag="emerging">Emerging</ClaimTag>
<ClaimTag tag="scenario" />
```

`verified` (accent), `emerging` (amber), `scenario` (muted). Tiny by design — attach it next to a stat or claim in captions/credibility cards, not on big headlines.
