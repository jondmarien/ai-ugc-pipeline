# AI Cybersecurity Carousel Assets

This package contains the visual demo assets generated from the AI cybersecurity UGC carousel skill.

## Contents

| Folder/File | Purpose |
| --- | --- |
| `ready_to_post_1080x1350/cover_backgrounds/` | Three text-free cinematic cover backgrounds for later typography in Canva, Figma, or a social design tool. |
| `ready_to_post_1080x1350/finished_carousels/` | Three finished 8-slide carousel sets, 24 slides total, exported at 1080 × 1350 px. |
| `contact_sheets/` | Quick visual review sheets for the cover backgrounds and each carousel. |
| `prompts/production_brief.md` | The prompts and exact text blocks used to generate the visual assets. |
| `asset_manifest.json` | Counts, dimensions, and package metadata. |
| `validation_notes.md` | Visual QA notes and caveats from review. |

## Publishing Note

The cover backgrounds are safer for production because they are text-free and can be paired with exact editable typography. The finished carousel slides are useful as a demo and may be postable, but they include AI-rendered text, so they should receive a final spelling and brand review before publishing.
