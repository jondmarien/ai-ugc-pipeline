An open-source AI agent quietly became one of the most-used agents in the world, and outside dev circles almost nobody noticed.

Hermes Agent, from Nous Research, launched February 25 and has blown past 180,000 GitHub stars since. It's MIT-licensed and runs on your own box. The hook is the learning loop: finish a hard task and it writes a reusable skill from the conversation, then loads that skill next time instead of relearning. It keeps curating those skills as you go, so it gets sharper the longer you use it. That's the genuinely cool part.

Here's the part I keep flagging. This is an always-on agent with a terminal, web access, and your API keys. That's a real new surface, not a reason to panic. Run it in one of its sandbox backends, give it least-privilege keys, log what it touches, and treat community-shared skills like untrusted code, because skills execute.

And 'self-improving' isn't the sci-fi version. It curates skill files; it doesn't rewrite its own brain. Useful and real, minus the hype.
